VCF to ANNOVAR Converter

This module will help you convert one VCF file to one CSV file with Annovar variant annotations. Please use this module if you have already generated one VCF per person using WebSeq’s FASTQ to VCF module.

If you have 2 VCFs from the same person (e.g. one with SNV and another with Indels), please use this module. This module will merge the 2 VCFs and then generate a single CSV file with Annovar variant annotations.
