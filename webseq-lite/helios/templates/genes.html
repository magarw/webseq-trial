This page allows you to pull out variants from a set of user-defined genes of interest. First, you need to create a .csv file with your genes of interest in the first column. Note that you must use the official gene symbols , and you must add the column header Gene.refGene

Next, click the Upload Files button and select the .csv file containing variants in the patient/individual. This file must be in the format that is generated by our VCF to Annovar CSV module. It may also be filtered by allele frequency, if you like.

Finally, click the Upload Files button again, and this time select the .csv file containing your genes of interest. Click submit.
