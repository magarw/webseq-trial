This page supports filtering of variants according to their reported minor allele frequencies in the gnomAD database.

The user may choose different types of reference data within gnomAD, for example all gnomAD exome data (gnomAD_exome_ALL), or population-specific reference data, for example Ashkenazi Jewish individuals’ exome data (gnomAD_exome_ASJ).

Please refer to the gnomAD website (https://gnomad.broadinstitute.org) for further information about the gnomAD database.


When Filtering by Allele Frequency is finished, download the results to archive a copy, and then push the results forward to the next step in this module (Modes of Inheritance).
